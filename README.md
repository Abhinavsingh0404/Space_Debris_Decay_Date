# Space Debris Decay Prediction

A machine learning pipeline to predict space debris decay dates using orbital parameters.

## Features

- Data loading and preprocessing for space debris datasets
- Feature engineering for orbital mechanics
- Multiple model comparison (Random Forest, XGBoost, etc.)
- Model performance evaluation and visualization
- Complete pipeline with logging and error handling
